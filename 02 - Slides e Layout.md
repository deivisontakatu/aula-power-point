# 📐 PowerPoint — Slides e Layout

## 📌 Introdução

O layout define a organização dos elementos dentro do slide.

---

## 🧱 Layout

Modelo de organização do slide.

---

## 📑 Tipos de Layout

* Título
* Título e conteúdo
* Somente conteúdo

---

## 🧾 Organização

Define:

* Posição do texto
* Espaço para imagens

---

## 🔄 Alterar Layout

Pode ser modificado a qualquer momento.

---

## 📊 Estrutura Visual

* Título
* Conteúdo
* Espaço organizado

---

## ⚠️ Pegadinhas

* Layout ≠ conteúdo
* Layout não altera dados
* Layout organiza, não cria

---

## 🧠 Boas Práticas

* Usar layout adequado
* Evitar sobrecarga
* Manter padrão visual

---

## 📌 Conclusão

O layout organiza a apresentação e melhora a leitura.
